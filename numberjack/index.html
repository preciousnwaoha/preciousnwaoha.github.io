About

Game play

Enter the number of persons that will play NumberJack in the input box and click/tap on the person icon to select computers too, a max only 12 players are allowed in NumberJack.

Type in a maximum number between 21 and 100 for 'Max Num' and the click/tap 'start Game' to start.

The Goal of the each player is to try to get the highest score on the score board after an agreed number of rounds.

A round ends when a player wins. To start a new round, when a player wins tap 'Next Round'.

Story

I wanted to learn javascript as my first programming language, so is went on to our trusty YouTube and found a tutorial from the CEO of Scrimba.com.

In the video he built a simple web 'game' that generated randam numbers when a button is clicked and summed the numbers with each click.
It was easy, too easy!, so i challenged myself to build a full funtcioning playable game that could actually be enjoyed.

So here we are...

I'm still improving the game and open to suggestions.
